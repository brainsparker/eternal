# Eternal — The Permanent Collection

A self-betraying generative art gallery.

The machine endlessly mints brand-new, "irreplaceable" 1-of-1 masterpieces — frames each one like a museum piece, swears on the placard that it's *permanent* — and then, a few seconds later, quietly deaccessions it into the dark and starts over.

A counter keeps score: **MINTED** climbs forever; **KEPT** stays at 0. There's a "Save this one forever" button. It can't.

You don't have to do anything. It betrays itself on a timer.

→ **https://brainsparker.github.io/Eternal/**

## Running it

It's a single, self-contained HTML file. Open `index.html` in any modern browser — no build step, no dependencies (fonts load from the Google Fonts CDN).
